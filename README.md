## 📦 Instalación y Uso

Sigue estos pasos para correr el proyecto localmente:

```bash
# Clona el repositorio
git clone https://github.com/JuanM-8/Portafolio_Rect

# Entra en la carpeta
cd Portafolio_Rect

# Instala las dependencias
npm install

# Inicia el servidor de desarrollo
npm run dev
```
